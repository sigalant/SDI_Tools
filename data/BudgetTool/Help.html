SDI_Tools

Budget Formatting Tool

How to Use

When the application is opened, you will be greeted by a window with three buttons and two text boxes. The textbox under "Markup Percent" will be used in the formatted budget sheet to calculate the total cost for a project. The text box under "Enter Good Through Date:" is also used in the formatted budget sheet to indicate when this formatted budget sheet is valid.

Both the "Select Input File" button and the "Select Output Folder" button will open a file explorer window when clicked. The "Select Input File" button expects that you select an Excel file exported from AutoQuotes which contains the following headers: "ItemNo", "Qty", "Category", "Sell", "Remarks", "Spec", "Model", "Unit", "SellTotal". The "Select Output Folder" button expects that you select the location you want this application to save the formatted Excel sheet to. Both the paths for the input file and output folder will be displayed in the window after they have been selected.

When you are ready to create the formatted budget file (input/output location set, and text boxes filled), you can click the button labeled "format file".

The application will then use the information from the supplied AQ export to create a formatted Budget sheet at the provided location. During formatting, the application will skip over any rows which has a non-white background (r, g, or b values less that 0xEF), sets the sell column of NIC items to 'NIC', sets qty of SpareNo items to '-' and sell to empty, sets qty of custom fab items which use ft as units to 1, and sums the SellTotal of all items at the bottom of the sheet before applying the markup percent.

[Warning] Possible errors could occur if:

- the first row of the exported AQ Excel file is not populated with the headers of the columns. 
- the logo located in the "AutoQuotes Budget Script" folder is removed, or replaced with an image with a different name
- Any of the expected headers are missing from the input file
