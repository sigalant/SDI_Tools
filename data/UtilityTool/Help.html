SDI_Tools

Utility Loads Formatting Tool

How to Use

When the application is first opened, you will be greeted by a window with five buttons and three text boxes. The buttons labeled "select input file" and "select output folder" will open a file explorer window. The "select input file" button expects that you select the Utility Loads excel export from AutoQuotes (must contain the following columns for imperial units: "KW", "GPH", "BTUS", "EXH CFM", "SUPPLY CFM", "VOLTS", "PH", "HEAT REJECTION", "NO", "QTY" or for metric units: "AMPS", "VOLTS", "PH", "LPH", "GAS KW", "EXH (M^3/H)", "SUPPLY (M^3/H)", "HEAT REJECTION WATTS", "NO", "QTY"), and the "select output folder" button expects that you select the location you want the application to save the formatted Utility Loads sheet to. The three text boxes start filled with the numbers: 120, 208, and 480. These are the acceptable voltages. Any voltage on the formatted sheet that isn't one of these numbers will be highlighted in yellow for later review. You can change the numbers in the text boxes, but they must remain as whole numbers. You can also increase the number of text boxes, and therefore the number of acceptable voltages (up to 5), by clicking the "+" button. Likewise you can reduce the number of text boxes, and number of acceptable voltages, by clicking the "-" button (minimum of 1).

When you are ready to create the formatted utility loads sheet (filled acceptable voltages and selected input/output locations). You can click the "format file" button. This button will create a new file where the following columns are summed for imperial units: "KW", "GPH", "BTUS", "EXH CFM", "SUPPLY CFM", "HEAT REJECTION" or for metric units: "AMPS","LPH","GAS KW", "EXH (M^3/H)", "SUPPLY (M^3/H)", "HEAT REJECTION WATTS". KW are calculated for each item if an "AMPS" column exists, otherwise "KW" values are copied over. Items with multiple electrical utilities are separated into different rows for each utility, and any utility which is summed and has a quantity is multiplied by that quantity.

[Warning] Possible errors:

- Headers are extracted from the top row of the excel sheet, so if the headers are located on another row, they might be recorded as missing. 
- All headers must exist in the input file or the application won't produce an output file.
